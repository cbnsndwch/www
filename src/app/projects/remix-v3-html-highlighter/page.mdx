**Remix v3 HTML Highlighter** is the definitive VS Code extension for developers building modern web interfaces with Remix and tagged templates. Originally a fork of `comment-tagged-templates`, this project has evolved to offer a "zero-config" experience, removing the need for manual comment markers while focusing exclusively on high-performance HTML syntax highlighting.

By stripping away support for miscellaneous languages and focusing strictly on the Remix ecosystem, we provide a stable, lean, and conflict-free editor experience. Whether you're using legacy comment-style tags or the new bare `html` tagged templates, this extension ensures your code is beautiful and readable from day one.
