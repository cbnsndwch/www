So the [@portainerio](https://x.com/portainerio) REST API docs are, ahem, **not great** (oh Swagger, how I love you!).

I kept wasting tokens and time fighting the agent but because I'm <s>stubborn</s> determined I had to make it work. So I shaved a yak and made an LLM + human-friendly docs site I can point my clanker to.

Absolute shoutout to [@fuma_nama](https://x.com/fuma_nama) for the UX bits and the OpenAPI interop (Custom Generators FTW!). The silicon citizen (ha!) is now happy because it can peruse the MDX contents and I am delighted that I get to save my [@GitHubCopilot](https://x.com/GitHubCopilot) premium requests to do actual work.
